# ID Document Forgery Detection

This project is a prototype system that analyzes an uploaded ID document image and generates a fraud risk report.

# Features

* Upload ID document image
* Image validation
* Blur detection for image quality
* OCR text extraction using Tesseract
* Structured fraud risk report

# Technologies Used

* Python
* Streamlit
* OpenCV
* Tesseract OCR
* Pillow
* NumPy

# How to Run

1. Install dependencies
   
   pip install -r requirements.txt

2. Run the application
   
   streamlit run app.py
